# randem
Random Emoji Printer

You can use this to print random emojis in your terminal.

```console
$ randem --help
Usage: randem [OPTIONS]

Options:
  -s, --seed <SEED>                    
  -i, --include-group <INCLUDE_GROUP>  
  -e, --exclude-group <EXCLUDE_GROUP>  
  -h, --help                           Print help
```
